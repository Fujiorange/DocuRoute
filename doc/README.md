## How to Regenerate

To regenerate these screenshots:

1. Ensure the Next.js development server is running:
   ```bash
   npm start
   ```

2. Run the screenshot script:
   ```bash
   npm run screenshot
   ```

The script uses Playwright to automatically capture screenshots of all pages.
